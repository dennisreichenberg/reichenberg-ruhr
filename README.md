# reichenberg-ruhr

Portfolio website and tool ecosystem for Dennis Reichenberg.

## Tools

| Tool | Description | Repo |
|------|-------------|------|
| ollama-commit | AI-generated Git commit messages via Ollama | [GitHub](https://github.com/dennisreichenberg/ollama-commit) |
| ollama-tui | Terminal UI for Ollama chats | [GitHub](https://github.com/dennisreichenberg/ollama-tui) |
| ollama-local-rag | Local RAG pipeline with Ollama | [GitHub](https://github.com/dennisreichenberg/ollama-local-rag) |
| ollama-proxy | Unified API router for Ollama/vLLM | [GitHub](https://github.com/dennisreichenberg/ollama-proxy) |
| ollama-monitor | Real-time TUI dashboard for Ollama inference | [GitHub](https://github.com/dennisreichenberg/ollama-monitor) |
| vllm-config | Hardware-aware vLLM configuration | [GitHub](https://github.com/dennisreichenberg/vllm-config) |
| vllm-monitor | Real-time dashboard for vLLM | [GitHub](https://github.com/dennisreichenberg/vllm-monitor) |
| model-manager | Local LLM model lifecycle manager | [GitHub](https://github.com/dennisreichenberg/model-manager) |
| llm-usage | Historical request and cost accounting across all LLM backends | [GitHub](https://github.com/dennisreichenberg/llm-usage) |

## Local development

```bash
npm install
npm run dev
```
